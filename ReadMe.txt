local_xlsx_file_parser.py python program demo following:

1. How to load existing xlsx file and parse through to capture data set using xlrd APIs.
2. Create a new xlsx file and populate dataset using xlswriter APIs.
3. Update existing xlsx file using openyxl python APIs.
4. Demo global variable usage, applying multiple business rules on multiple row dataset.

Some of data fields (rows and columnds) are hard coded. Attaching sample.xlsx files just to provide 
you an example of file which is being used for parsing.
